# UniCribs-Smart-Student-Rental-Platform
UniCribs is a student-focused rental platform designed to simplify accommodation discovery, ensure safety, and improve communication between students and landlords. This project highlights the core concept, market research, value proposition, business model, and features of UniCribs.

# Problem Statement

• Students find it difficult to access verified and safe accommodation
• Communication gaps create misunderstandings between students and landlords
• Existing platforms are general-purpose and time-consuming to navigate

# Solution Overview

UniCribs provides a student-centric rental ecosystem with:

• Verified property listings
• Secure transactions and smart filtering
• Direct communication with landlords
• Fast booking and simplified processes

# Core Feature

For Students:

• Search and filter verified rooms
• Chat directly with property owners
• Book securely with simplified payment flow

For Landlords:

• List units easily
• Manage inquiries and bookings
• Receive rental payments with a transparent fee system
